# Grove Review: Pilot Migration Issues

This note records whether the local Grove side materials clarify selected Work
Group, Work and Performance-candidate review items from the pilot migrations.

Sources checked:

- `side materials/Music collection/Grove/Britten-Dohnanyi.doc`

The Grove documents should be treated as authority evidence for manual review,

## Bruckner

Grove strongly supports grouping most flagged symphony version rows as multiple
Works inside one Work Group per symphony.

- Symphony No. 1: Grove distinguishes an early/original scherzo and adagio from
  a later second version. Keep one Work Group, with separate Work records for
  versioned states where the source identifies them.
- Symphony No. 2: Grove lists the 1872/1873 state, later revisions in 1873 and
  1876, and a second 1877 version revised in 1892. Keep one Work Group; version
  assignment is meaningful and should stay explicit.
- Symphony No. 3: Grove lists the 1873 work, a 1874 revision, a separate 1876
  Adagio/intermediate slow-movement state, and later versioning. The source-line
  Tidal candidates should remain candidates unless the recording label clearly
  identifies the version or movement.
- Symphony No. 4: the source rows are versioned states of one symphony family.
  Keep one Work Group with separate Works for 1874-76, 1878, 1880 and 1888
  where accepted.
- Symphony No. 5, 7, 8 and 9: Grove confirms meaningful version or completion
  language for the flagged rows. Candidate recordings should only become
  canonical Performances when their version assignment is clear.
- `4 Tantum ergo`: Grove lists WAB 41 as a set of four Tantum ergo settings,
  later revised in 1888. Treat this as a set/work boundary question, not as an
  ordinary single Work. The current candidate should remain non-canonical until
  the recording item can be mapped to the set or to an individual setting.

## Mahler

Grove clarifies the main flagged uncertainty.

- Symphony No. 10: Grove lists the work as incomplete and separately identifies
  a complete performing version by Deryck Cooke. A Yoel Gamzou realization is
  therefore not the generic Symphony No. 10 Work. It should be modelled as a
  completion/realisation Work inside the Symphony No. 10 Work Group, or remain a
  Performance recommendation candidate until that Work exists.
- Das klagende Lied: Grove confirms revisions in 1892-93 and 1898-99, with an
  original three-part version including `Waldmärchen`. This should be one Work
  Group with versioned Works if recommendations distinguish the version.
- Das himmlische Leben: Grove lists this as a song and also notes it as the text
  of Symphony No. 4 movement 4. Keep a separate song Work, with relationship
  evidence to Symphony No. 4 rather than merging it into the symphony Work.
- Die drei Pintos: Grove lists this under Mahler's performing
  editions/arrangements as reconstruction and augmentation of Weber's incomplete
  opera. This should not be a normal Mahler original Work Group; it needs an
  arrangement/completion policy decision.
- Der Trompeter von Säckingen: Grove lists it as incidental music. The local
  evidence supports an ordinary Work unless another source introduces a derived
  `Blumine` relationship.

## Stravinsky

Grove gives useful boundary evidence for most flagged rows.

- The Firebird: Grove lists the ballet and separately lists `Suite from The
  Firebird`. Treat the suite as a separate Work inside the same Work Group or as
  a derived Work related to the ballet family.
- Petrushka: Grove lists the original 1910-11 work and a 1946 revision, with
  revised publication in 1948. This supports one Work Group with separate Works
  for the original and revised versions. The source labels `1911 version` and
  `1947 version` should be normalized against Grove's 1910-11 / 1946 revision /
  1948 publication evidence.
- The Rite of Spring: Grove lists the 1911-13 work and a revised `Sacrificial
  Dance` in 1943/1945. This supports keeping revision evidence, but a complete
  recording should not be assigned to a special revised Work unless the recording
  identifies that version.
- Pulcinella: Grove lists the ballet and a `Suite from Pulcinella` separately.
  Treat the suite as a derived Work, not as the same Work as the ballet.
- Apollo and Le baiser de la fee: Grove lists the ballets as works; it also
  identifies `Divertimento` as an arrangement from `Le baiser de la fee`.
- Quatre etudes: Grove explicitly identifies this as an orchestral arrangement
  of earlier pieces. Treat as a derived Work.
- Scherzo a la russe: Grove lists both orchestral and jazz-band versions. This
  supports separate Works inside one Work Group when both are represented.
- Les Noces: Grove distinguishes incomplete draft history from completion in
  1921-23. Keep version evidence; only split if recommendations require it.

## Beethoven

Grove clarifies several flagged rows.

- Piano Concertos Nos. 1 and 2: Grove lists them as distinct concerto Works.
  The report flag is likely a false positive from numbering/title matching.
- Beethoven's arrangement of Op. 61 for piano: Grove explicitly lists this as an
  arrangement of the Violin Concerto. It should be a separate Work in the same
  Work Group as the Violin Concerto, not a Performance variant.
- Choral Fantasy: Grove lists it as a distinct work, while also noting that the
  melody of `Gegenliebe` was later used in it. Keep as its own Work Group, with
  possible relationship metadata to `Gegenliebe`; do not merge them.
- Schneider Kakadu variations: Grove confirms an early date and a 1816 revision.
  Keep one Work Group; split into versioned Works only if performance sources
  distinguish versions.
- Allemande WoO 81: Grove confirms revision in 1822. Keep one Work Group; treat
  version assignment as relevant only if recordings distinguish it.
- Symphonies Nos. 1, 2, 3, 4, 6, 7 and 8: Grove lists these as distinct
  ordinary symphony Works. The report flags are likely false positives unless
  another source identifies arrangements or revisions for a specific row.

## Hindemith

The local `Hindemith.doc` Grove export is a short two-page works extract and
does not include the flagged items `Cardillac`, `Neues vom Tage`, `Herodiade`,
`Das Marienleben` or `Quintet`.

It therefore does not resolve those review lines. Those items need another
authority source or a fuller Grove export.

## Bach

The local Grove set does not appear to contain a usable J.S. Bach works list for
the flagged BWV rows. The Mahler Grove entry does identify Mahler's `Suite aus
den Orchesterwerken` as an arrangement drawing from Bach orchestral suites.

The Bach review items therefore remain unresolved by the local Grove documents:

- Brandenburg Concerto No. 5 early/revised version handling;
- BWV 1044 / BWV 1052 version and derivation boundaries;
- BWV 1059 fragment/reconstruction handling;
- Orchestral Suite version or arrangement boundaries;
- `Suite for Organ, Harpsichord & Orchestra` arrangement attribution.

These need a fuller Bach authority source before canonical Performance
promotion.
