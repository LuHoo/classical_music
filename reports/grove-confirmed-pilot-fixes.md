# Grove-Confirmed Pilot Fixes

This report records canonical YAML changes made from Grove evidence after the
pilot migration review.

## Scope

Applied only when the local Grove materials clearly supported the modelling
decision.

Unresolved cases remain in the migration review issues and in
`reports/grove-review-pilot-migration-issues.md`.

## Applied Changes

- Bruckner symphony versions now share one Work Group per symphony for the
  Grove-confirmed version families:
  - Symphony No. 1;
  - Symphony No. 2;
  - Symphony No. 3;
  - Symphony No. 4;
  - Symphony No. 5;
  - Symphony No. 7;
  - Symphony No. 8.
- Duplicate Work Group files for those Bruckner version rows were removed.
- Beethoven's piano arrangement of Op. 61 now belongs to the same Work Group as
  the Violin Concerto in D major, Op. 61.
- Stravinsky's `Suite from Pulcinella` now belongs to the `Pulcinella` Work
  Group as a derived Work.
- Stravinsky's `Divertimento` now belongs to the `Le Baiser de la fee` Work
  Group as a derived Work.
- Stravinsky's source-labelled `1947 version` Petrushka Performance now points
  to a separate Work for the Grove-supported 1946 revision / 1948 revised
  publication evidence.

## Not Applied

- Hindemith fixes: local Grove evidence was insufficient.
- Bach fixes: the local Grove folder did not include a usable J.S. Bach worklist
  for the flagged BWV rows.
- Bruckner candidate Performances whose recording version remains unclear:
  these remain outside any additional automatic promotion.
- Mahler Symphony No. 10 / Gamzou: left for a completion-realisation Work policy
  decision rather than guessing the canonical Work shape.
