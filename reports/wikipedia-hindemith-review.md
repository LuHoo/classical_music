# Wikipedia Review: Hindemith Pilot Issues

Source checked:

- <https://en.wikipedia.org/wiki/List_of_compositions_by_Paul_Hindemith>

The page was used as review evidence for the Hindemith pilot issues, not as an
automatic migration source.

## Applied Changes

- `Hérodiade` source rows now share one Work Group. Wikipedia lists it as one
  ballet with Martha Graham; the source distinction between version without
  words and version with recitation is treated as performance/profile evidence,
  not a separate artistic-family boundary.
- `Cardillac` records now note that Wikipedia confirms a revised version
  premiered in 1952.
- `Neues vom Tage` records now note that Wikipedia confirms a revised version
  premiered in 1954.
- `Das Marienleben` records now note that Wikipedia confirms two versions,
  1923 and 1948, plus a composer orchestration of six songs.
- `Quintet`, Op. 30 records now note that Wikipedia confirms revision in 1954.

## Not Applied

- No extra versioned Work was created for `Cardillac`, `Neues vom Tage`,
  `Das Marienleben` or `Quintet`, Op. 30. The pilot source does not assign a
  current recommendation to a specific version, so creating specific version
  Works would be premature.
